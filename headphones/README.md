# Headphones Website
## Project Overview
This project is a visually appealing and responsive webpage for a headphones brand. It is based on a Figma design and focuses on implementing pixel-perfect styling using HTML and CSS. The project includes interactions, hover effects, and a mobile-friendly design.

## Features
- **Responsive Design:** Automatically switches to a mobile-friendly layout when the screen width is 480px or less.
- **Hover Effects:**
  - Links change color to `#FF6565` when hovered or active.
  - Buttons change opacity to `0.9` on hover or active states.
- **Centered Content:** The main content is centered on the page with a maximum width of `1000px`.
- **Typography:** Implements custom fonts (`Source Sans Pro` and `Spin Cycle OT`) for a professional and consistent look.

## Technical Specifications
- **Languages Used:**
  - HTML
  - CSS
- **Responsive Breakpoint:** 480px.
- **Interactions:**
  - Links: `#FF6565` color on hover/active.
  - Buttons: `opacity: 0.9` on hover/active.
- **Design Alignment:** Matches the Figma design details as closely as possible.
